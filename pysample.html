Domain 1: Inputs and Outputs

In this domain, you'll learn to get information from the user and display results using print() and input().

🎯 Objective

Use input and output functions in Python to build basic interactive programs.

📝 Exercise

# Ask for the user's name and greet them
name = input("What is your name? ")
print("Hello, " + name + "!")

✅ Task

  1. Run the code above.
  2. Modify it to also ask for age and favorite color.
  3. Print a sentence that uses all three answers.
