Lesson 10: Output

In this lesson, you'll learn how to use the print() function to send messages to the user. This is your main way of showing results or feedback in a Python program.

🎯 Learning Target: I can display output to the screen using print().

💡 Example:

print("Hello, world!")
print("The answer is", 42)
print("Let's learn Python together!")

✏️ Your Turn:

  1. Write a program that prints a greeting to the user.
  2. Print your favorite quote.
  3. Print a message that includes both text and numbers.

Lesson 11: Input

You're ready to talk to your computer! The input() function allows users to enter information while the program runs.

🎯 Learning Target: I can get input from a user using input().

💡 Example:

name = input("What's your name? ")
print("Hello,", name)

✏️ Your Turn:

  1. Ask the user for their favorite food and print a message about it.
  2. Ask the user where they are from and respond with a greeting.

Lesson 12: Comments

Comments are notes you leave for yourself (or other coders). Python ignores them when your code runs.

🎯 Learning Target: I can write comments using # in my code.

💡 Example:

# This line says hello to the user
print("Hello!")

# The next line prints the user's favorite number
print("I like the number", 7)

✏️ Your Turn:

  1. Add comments above two different print() statements.
  2. Write a short program with at least three lines of comments explaining what’s happening.

Lesson 13: Escape Characters

Sometimes you want to include special characters in your text like quotes or line breaks. That’s where escape characters come in!

🎯 Learning Target: I can use escape characters like \n and \t to format strings.

💡 Example:

print("She said, \"Python is fun!\"")
print("First line\nSecond line")
print("Item\tPrice")

✏️ Your Turn:

  1. Print a sentence with double quotes inside it.
  2. Use \n to print text on two lines.
  3. Use \t to space out items in a list.

Lesson 14: String Concatenation

You can combine strings using +. This is called concatenation. It’s great for building dynamic messages!

🎯 Learning Target: I can combine strings using +.

💡 Example:

first_name = "Ada"
last_name = "Lovelace"
full_name = first_name + " " + last_name
print("Hello,", full_name)

✏️ Your Turn:

  1. Ask the user for their first and last name and print their full name.
  2. Make a sentence using at least two string variables and +.

Lesson 15: String Repetition

Want to repeat a word or symbol several times? Python lets you multiply strings!

🎯 Learning Target: I can repeat a string using *.

💡 Example:

print("ha" * 3)     # prints 'hahaha'
print("-" * 10)     # prints '----------'

✏️ Your Turn:

  1. Print a funny sound repeated multiple times.
  2. Create a line break using symbols (like - or *).
