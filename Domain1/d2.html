Lesson 14: What is a Data Type?

Data types tell Python what kind of data we are working with. Common types include:

  • int - integers (whole numbers)
  • float - decimal numbers
  • str - strings (text)
  • bool - booleans (True or False)

Lesson 15: Python is Strongly and Dynamically Typed

Python automatically figures out the data type, but it doesn't let you mix incompatible types.

What happens when we try to add an integer and a string?

x = 5
y = "7"
print(x + y)  # ❌ This will cause a TypeError

Fix it by converting the string to an integer:

print(x + int(y))  # ✅ 12

Lesson 16: What is a Variable?

Variables are names that store values. You can use them to store data and refer back to it later.

name = "Luna"
age = 14
gpa = 3.8
print(name, "is", age, "years old with a GPA of", gpa)

Lesson 17: Variable Naming Rules

Variable names:

  • Must start with a letter or underscore
  • Can include letters, numbers, and underscores
  • Cannot be a Python keyword like if, else, while, etc.
first_name = "Alex"
_lastName = "Rivera"
score2 = 98
