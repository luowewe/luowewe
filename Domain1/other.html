Lesson 1: What is Python?

Python is a general-purpose, high-level programming language. It's known for readability and versatility.

Lesson 2: Python Syntax and Structure

Understand indentation, statements, variables, and data types.

Lesson 3: IDEs and Tools

Python can be run in IDLE, VS Code, online editors like Replit and Trinket, or in a terminal using the `python` command.

Lesson 4: Running and Debugging Code

Debugging includes checking syntax, logic, and runtime errors.

Lesson 5: Variables and Data Types

Create and modify variables. Use type() to check types.

name = "Luna" age = 17 gpa = 3.7 print(type(name), type(age), type(gpa))

Lesson 6: User Input

Use input() to gather and use data from users.

name = input("What's your name? ") color = input("Favorite color? ") print(f"Hi {name}, your favorite color is {color}!")

Lesson 7: Math Operations

Perform basic math and use user inputs in calculations.

a = int(input("Enter a number: ")) b = int(input("Enter another: ")) print("Sum:", a + b) print("Product:", a * b)

Lesson 8: String Manipulation

Practice with .upper(), .lower(), slicing, etc.

msg = " Hello World! " print(msg.strip().upper()) print(msg[1:6])

Lesson 9: Final Practice Challenge

Create a program that uses input, math, and string formatting.

name = input("Name: ") age = int(input("Age: ")) years = 100 - age print(f"{name}, you'll be 100 in {years} years!")
