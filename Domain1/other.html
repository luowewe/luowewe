Domain 1: Create the Variable

Learn how to declare, assign, and use variables in Python programs.

Lesson 1: What is a Variable?

A variable is a container for storing data values. In Python, you assign a value to a variable using the = sign.

Example:

name = "Alex"
age = 14
is_student = True

In the code above, name is a string, age is an integer, and is_student is a boolean.

Lesson 2: Variable Types

Python has different types of data you can store in variables. The most common types are: str (string), int (integer), float (decimal numbers), and bool (true/false).

🎯 Learning Target

📘 Example


name = "Jordan"       # string
age = 15              # integer
height = 5.7          # float
is_student = True     # boolean

print(name)
print(age)
print(height)
print(is_student)

Lesson 3: Variable Assignment

In Python, you can assign values to variables using the = operator. This allows you to store and update data as needed.

🎯 Learning Target

I can assign and reassign values to variables in Python.

📘 Example


x = 10
print(x)  # Outputs: 10

x = 20
print(x)  # Outputs: 20

Lesson 4: Variable Naming Rules

Python has specific rules for naming variables. Understanding these rules helps in writing clear and error-free code.

🎯 Learning Target

I can identify valid and invalid variable names in Python.

📘 Example


# Valid variable names
user_name = "Alice"
age = 25
_is_active = True

# Invalid variable names
2nd_place = "Bob"     # Starts with a number
user-name = "Charlie" # Contains a hyphen
class = "Physics"     # 'class' is a reserved keyword

Lesson 5: Combining Variables

You can combine variables using operations like addition or concatenation, depending on their data types.

🎯 Learning Target

I can combine variables of different types appropriately in Python.

📘 Example


# String concatenation
first_name = "John"
last_name = "Doe"
full_name = first_name + " " + last_name
print(full_name)  # Outputs: John Doe

# Combining string and integer
age = 30
print("Age: " + str(age))  # Outputs: Age: 30
