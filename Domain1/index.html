Domain 1: Create the Variable

Use the lessons and code editors to practice each concept. No need to refer back to the workbook!

🎯 Learning Goals

  • Identify and use appropriate data types in Python (str, int, float, bool).
  • Declare and assign values to variables.
  • Use descriptive variable names following Python naming conventions.
  • Concatenate and format variables for output.

📘 What is a Variable?

A variable stores data for your program. You assign a value to it using =.

name = "Alex"
age = 14

You can now use name and age in your code!

Exercise 1.1 – Variable Types

Declare at least one variable of each type: string, integer, float, and boolean. Print their values.

Exercise 1.2 – Variable Assignment

Create a variable, assign it a value, print it, change the value, and print again to observe the update.

Exercise 1.3 – Combining Variables

Use string concatenation and type casting to print a sentence with both text and numbers.

Exercise 1.4 – Naming Rules

List examples of valid and invalid variable names. Then write a program using descriptive, snake_case variable names.

💡 Challenge – Favorite Things

Write a program using at least three variable types to describe your favorite things. Example: food, pets, whether you love summer, etc.
