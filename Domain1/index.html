Domain 1: Create the Variable

Use the lessons and embedded code editors to practice each concept below. Refer to the workbook for examples and instructions.

Learning Goals

  • Identify and use appropriate data types in Python (str, int, float, bool).
  • Declare and assign values to variables.
  • Use descriptive variable names following Python naming conventions.
  • Concatenate and format variables for output.

Exercise 1.1 – Variable Types

Write a Python program that declares at least one variable of each type: string, integer, float, and boolean. Print each variable’s value.

Exercise 1.2 – Variable Assignment

Declare a variable, assign a value, print it, then reassign a new value and print again to observe the change.

Exercise 1.3 – Combining Variables

Create a program that uses string concatenation and type casting to output a sentence using both text and numbers.

Exercise 1.4 – Variable Naming Rules

List valid and invalid variable names. Then, write code using proper naming conventions (snake_case, descriptive names).

Challenge – Favorite Things

Write a program using at least three variable types to describe your favorite things (like favorite food, number of pets, is summer your favorite season, etc.).

Summary

In this domain, you practiced creating variables of different data types, reassigning them, combining them, and following naming conventions. Understanding these foundations is key to writing readable and functional Python code.
